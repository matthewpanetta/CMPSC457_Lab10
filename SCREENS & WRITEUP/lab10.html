Instructions

In this lab I needed to:
  • Draw a fish using a bezier curve
  • Use arrow keys to rotate around scene
  • Animate the fish's tail

My Implementation

  • I used the control points given to me, and copied them to represent the right side of the fish.
  • A special key function allows the user to rotate the fish around the scene by using the arrow keys.
  • Press the home key to reset the angle.
  • The fish is initially rotated at an angle so it is not viewed straight on. (You can change this in the code)
  • I drew water (a cube), to simulate an ocean. The fish can rotate around in the water.
  • The fish's tail swings back and forth.
